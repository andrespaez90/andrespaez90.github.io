BitMonitor Support

Welcome to the BitMonitor support page! If you encounter any issues or have suggestions for improving BitMonitor, we're here to help. Follow the steps below to report an issue or request support.

How to Open an Issue

  1. Visit the GitHub Issues Page:

    Go to our GitHub Issues Page to view existing issues and report new ones.

  2. Create a New Issue:

    On the GitHub Issues Page, click the "New Issue" button located on the right side of the page.

  3. Fill Out the Issue Form:
    • Title: Provide a concise and descriptive title for the issue.
    • Description: Describe the issue in detail. Include steps to reproduce the problem, expected behavior, and any additional information that may help us understand the issue.
    • Screenshots/Attachments: If applicable, attach screenshots or files that illustrate the problem.
  4. Submit the Issue:

    Once you have completed the form, click the "Submit new issue" button to report your issue.

What Happens Next?

After you submit an issue, our team will review it and respond as soon as possible. We may request additional information if needed. You can track the progress of your issue directly on the GitHub Issues Page.
